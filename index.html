redirecting you to a random mirror

This is a method of trying to keep mesacomplex unblocked. this works basically like this: we have a bunch of different identical versions of mesacomplex, and whenever you go to mesacomplex.tk, you are taken to one of these versions. that way, if one of these versions gets blocked, we can simply remove it from the list of random redirects and you would never notice!
